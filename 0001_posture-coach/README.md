# 🪑 Posture Coach

웹캠으로 **거북목·구부정한 자세**를 실시간 감지해서, 자세가 무너지면 **데스크톱 알림 + 비프음**으로 알려주는 Chrome 확장 프로그램입니다. 감지는 백그라운드 **Offscreen 문서**에서 돌기 때문에 **사이드 패널을 닫아도 계속 모니터링**됩니다.

> 🔒 **프라이버시**: 카메라 영상과 자세 데이터(랜드마크)는 전부 이 기기 안(브라우저 내부)에서만 처리됩니다. 어떤 영상도 서버로 전송되지 않으며, 네트워크 통신 자체가 없습니다.

## 작동 방식

- **MediaPipe Pose Landmarker**(로컬 동봉)로 웹캠 영상에서 어깨·귀·코 위치를 추정합니다.
- "바른 자세"를 한 번 **보정(calibrate)** 해두면, 그 기준 대비 편차로 자세 점수(0~100)를 계산합니다.
- 점수가 임계치 아래로 **설정한 시간만큼 지속**되면 경고를 발사합니다.

### 감지 모드

| 모드 | 보는 방향 | 감지하는 것 |
| --- | --- | --- |
| **정면** | 카메라를 마주볼 때 | 구부정함(목 길이 감소), 고개 처짐, 화면에 너무 가까이, 좌우 기울기 |
| **측면(거북목)** | 옆모습일 때 | **두개척추각** — 귀가 어깨보다 앞으로 나올수록 거북목으로 판정 |
| **자동** | — | 어깨 너비/가시성으로 방향을 스스로 판단해 정면·측면을 전환 |

> 거북목은 본질적으로 **옆모습** 지표라 측면 모드가 가장 정확합니다. 모드별로 보정 기준이 따로 저장되니, 측면 모드는 옆으로 앉은 상태에서 한 번 더 보정해 주세요.

## 설치 (개발자 모드)

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장 프로그램 로드** 클릭 → 이 `posture-coach/` 폴더 선택
4. 툴바의 🪑 아이콘 클릭 → 사이드 패널이 열립니다

## 사용법

1. 사이드 패널에서 **모니터링 시작** 클릭 → (최초 1회) 카메라 권한 승인
   - 권한 팝업이 안 뜨면 **"카메라가 안 켜지나요? → 탭에서 권한 허용하기"** 버튼으로 허용
2. 스켈레톤이 움직이기 시작하면, **바른 자세로 앉은 뒤** `바른 자세 보정` 클릭
3. 이제 자세가 무너지면 점수가 떨어지고, 일정 시간 지속되면 비프음과 알림이 울립니다
4. **사이드 패널을 닫아도 백그라운드에서 계속 감지·경고**합니다. 멈추려면 패널을 다시 열어 **정지**를 누르세요

### 설정

| 항목 | 설명 |
| --- | --- |
| **민감도** | 높을수록 작은 흐트러짐에도 점수가 크게 떨어집니다 |
| **경고까지 지속 시간** | 나쁜 자세가 이 시간만큼 지속돼야 첫 경고가 나갑니다 (2~20초) |
| **반복 알림 간격** | 나쁜 자세가 유지되는 동안 이 간격마다 경고를 반복합니다 (3~60초). 자세를 펴면 즉시 멈춥니다 |
| **비프음** | 경고 시 소리 재생 on/off |
| **데스크톱 알림** | 경고 시 Chrome 알림 on/off |
| **화면 붉게 표시** | 경고 시 현재 보고 있는 탭에 빨간 맥동 테두리 + 메시지를 띄움 (보호 페이지 `chrome://` 등에선 동작 안 함). 자세를 펴면 사라집니다 |

> 경고가 켜지면 툴바 아이콘에 빨간 `!` 배지도 함께 표시되고, 자세를 펴면 모두 사라집니다.

설정과 보정 기준은 `chrome.storage.local`에 저장되어 패널을 닫았다 열어도 유지됩니다.

## 파일 구조

```
manifest.json        MV3 매니페스트 (sidePanel/notifications/storage/offscreen 권한)
background.js        서비스 워커 — 사이드 패널 열기, offscreen 생성/종료, 알림 발사(쿨다운 30초)
offscreen.html/js    백그라운드 감지 엔진 — 카메라 + 포즈 추정 + 점수/경고 (패널 닫혀도 동작)
sidepanel.html/css   사이드 패널 UI (스켈레톤·점수·통계 표시, 명령 전송)
sidepanel.js         UI 컨트롤러 — offscreen과 메시지로 통신
permission.html/js   카메라 권한 허용 전용 페이지 (탭에서 열림)
lib/
  vision_bundle.mjs  MediaPipe tasks-vision (로컬 동봉)
  wasm/              MediaPipe WASM 런타임
models/
  pose_landmarker_lite.task   포즈 추정 모델 (~5.8MB, lite)
icons/               아이콘 16/48/128
```

### 구조

- **감지(offscreen)**: 숨겨진 백그라운드 문서가 카메라와 MediaPipe 추정을 전담. 사이드 패널을 닫아도 살아 있어 계속 감지·경고함. (숨김 문서라 `requestAnimationFrame`이 throttle될 수 있어 `setInterval` 루프를 쓰고, 경고 지속 판정은 실시간 타임스탬프로 처리)
- **UI(side panel)**: offscreen이 보내는 상태/랜드마크를 받아 스켈레톤·점수·통계를 그리고, 시작/정지·보정·설정 명령을 보냄.
- **설정/보정 기준**은 `chrome.storage.local`에 저장되어 두 컨텍스트가 공유하며, offscreen은 변경을 즉시 구독함.

> ⚠️ 모니터링 중에는 **카메라 표시등이 계속 켜져 있습니다**(백그라운드 감지 중이라는 뜻). 끄려면 사이드 패널에서 **정지**를 누르세요. 브라우저를 새로 켜면 모니터링은 꺼진 상태로 시작합니다.

## 메모

- 모델은 가벼운 `lite` 버전입니다. 정확도가 부족하면 `pose_landmarker_full.task`로 교체할 수 있습니다.
- 정면 웹캠 기준으로 동작하므로, 옆모습보다는 **얼굴과 양 어깨가 모두 보이는 정면**에서 가장 정확합니다.
- 알림은 자세가 좋은 상태로 회복된 뒤 다시 무너지면 재발사되며, 연속 알림 폭탄을 막기 위해 최소 30초 간격이 적용됩니다.
